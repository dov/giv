# Description

giv is a cross platform image and hierarchical vector viewer based. It was designed especially for scientific vision and computational geometry and has support for drawing vector graphics on top of the image.

# Release notes

Release notes for version 0.9.32, 2020-07-20 Mon

New features:

* Added plugin for loading ora (OpenRaster) files.
* Larger default file for high resolution displays.
* Fixed bug in offset between pixels and grid lines when pixel grid is shown.
